# Todo Application
Todo application developed using JSP, Servlet, JDBC and MySQL Database.

## Following features
- Add Reminders
- Edit Reminders
- List Reminders
- Delete Reminders
- User Login
- User Signup
- Responsive layout

## Technology stack
- Java 8+
- JSP and Servlet
- JDBC
- MySQL database
- Eclipse IDE
- Tomcat server 8+
